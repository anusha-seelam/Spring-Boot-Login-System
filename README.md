# 🔐 Spring Boot Login System

## 📌 Overview

This project is a simple Login System developed using Spring Boot. It demonstrates how to create web pages using Spring MVC, configure Spring Security, and navigate between login and home pages.

---

## 🚀 Features

- User Login Page
- Home Page
- Spring MVC Controller
- Spring Security Configuration
- HTML Templates using Thymeleaf
- Maven Project Structure

---

## 🛠️ Technologies Used

- Java
- Spring Boot
- Spring MVC
- Spring Security
- Thymeleaf
- Maven
- Git
- GitHub

## 🔗 Endpoints

| HTTP Method | URL | Description |
|-------------|-----|-------------|
| GET | `/login` | Displays the login page |
| GET | `/home` | Displays the home page |

---

## ▶️ How to Run

1. Clone the repository.

```bash
git clone https://github.com/anusha-seelam/Spring-Boot-Login-System.git
```

2. Open the project in VS Code or IntelliJ IDEA.

3. Build the project using Maven.

4. Run `LoginApplication.java`.

5. Open your browser and visit:

```
http://localhost:8080/login
```
